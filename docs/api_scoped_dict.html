inseminator.scoped_dict¶

class inseminator.scoped_dict.ScopedDict(parent_dict: Optional[inseminator.scoped_dict.ScopedDict[inseminator.scoped_dict.K, inseminator.scoped_dict.V]] = None)¶

Dictionary with ability to specify the parent dictionary.

It tries to resolved __getitem__ call using the underlying dictionary and if it can’t find the key and tries to search in its parent dictionaries.
